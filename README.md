# Repositorio de Ejercicios de C/C++

## Descripción

Este repositorio contiene una colección de ejercicios básicos en **C** y **C++**, desarrollados como parte de un proyecto personal para repasar y fortalecer mis conocimientos en estos lenguajes. Los ejercicios están diseñados para cubrir conceptos fundamentales y son ideales tanto para principiantes como para quienes desean practicar y refrescar sus habilidades.

### Objetivos del Proyecto
1. Practicar conceptos clave de C y C++, como:
   - Variables, tipos de datos y operadores.
   - Control de flujo (if, for, while, switch).
   - Funciones y estructuras.
   - Manejo de memoria y punteros.
   - Clases, objetos y programación orientada a objetos (C++).(Todavia no llegamos a esta parte)
2. Crear una base de referencia para estudios posteriores y la implementación de proyectos más complejos.
3. Compartir estos ejercicios para que otros puedan utilizarlos como material de repaso.
